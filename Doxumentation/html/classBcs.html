Detailed Description

Boundary Conditions

Abstract base class for implementations of different boundary conditions. The fields to which the boundary conditions are applied are those passed into the function apply, not those in the SimData class.

Application function.

Pure virtual function definition of the apply function. Function will apply the specified boundary condition to the given vectors. Most often the convervative vector, or a vector of the same size (e.g. source/flux vector), will be the intended vector to apply boundary conditions too. Primitive and auxilliary variables can also be applied to.
