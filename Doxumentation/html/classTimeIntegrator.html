Detailed Description

General form of the time integrator

Abstract base class for all types of time integrators. Integrators require data to operate on, model for the flux functions and the boundary conditions to apply one the step has been made

Step function.

Pure virtual function: every time integrator must be able to increment time forward by a single timestep, dt, given by the simData.
Parameters
*conspointer to conserved vector work array. Size is \(N_{cons}*N_x*N_y*N_z\)
*primspointer to primitive vector work array. Size is \(N_{prims}*N_x*N_y*N_z\)
*auxpointer to auxilliary vector work array. Size is \(N_{aux}*N_x*N_y*N_z\)
dtthe step size desired to move by. Defaults to the value in the Data class

Implemented in SSP3, SSP2, BackwardsRK2, RKSplit, and RK2.
