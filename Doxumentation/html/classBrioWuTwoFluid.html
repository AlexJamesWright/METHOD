Brio-Wu shock tube initial data for two-fluid model.

Few options for the set up. All set ups will place a discontinuity along some axis, which is defined by dir where (0, 1, 2) = (x, y, z). The symmetric 3D generalisation is also selected by default (setUp=0), but for the values used in Amano 2016 use setUp=1. Stores a pointer to the Data class for reference in its methods

Parameters
*dataPointer to Data class containing global simulation data
dirdirection in which to place the discontinuity. (0, 1, 2) = along (x, y, z) axis
setUpType of set up, 0=3D consistent, 1=Amano 1D case
